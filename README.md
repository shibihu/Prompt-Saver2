# 📝 Prompt Saver

**Prompt Saver** is a personal web application that allows you to store, edit, copy, and manage your "Text Prompts" (AI commands) quickly in one place[cite: 1]. You no longer need to open note applications or send messages to yourself in chat apps[cite: 1]! When you need to use a prompt, simply open this website, click to copy it, and use it immediately[cite: 1].

---

## ✨ Key Features

* **💾 Store Prompt:** Save and organize your frequently used text prompts into an orderly list[cite: 1].
* **📋 Quick Copy:** A one-click button to copy the prompt text for immediate use[cite: 1].
* **✏️ Edit Content:** Modify the text or details of your prompts at any time to fine-tune your commands[cite: 1].
* **❌ Quick Remove:** Easily delete old or unwanted prompts to keep your library clean[cite: 1].
* **🌐 Access Anywhere:** Access your prompt library anytime and anywhere just by opening this project's website[cite: 1].

---

## 🚀 How it works
1. **Adding Data (Add):** Type your desired prompt into the system and save it to your library[cite: 1].
2. **Using It (Use):** When you want to talk to an AI (such as ChatGPT, Gemini, or Claude), go to this website and click the **Copy** button[cite: 1]. The system will copy the text to your clipboard, making it ready to be pasted into your AI tools[cite: 1].
3. **Managing the Library (Manage):** You can select **Edit** to change the command, or press **Remove** to delete items you no longer use[cite: 1].
